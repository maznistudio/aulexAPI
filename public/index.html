POST /api/veo

Generate video using AI. Supports text-to-video and image-to-video modes.

Request Body

Parameter Type Description
prompt required string Description of the video to generate
mode optional string text-to-video (default) or frames-to-video
aspectRatio optional string landscape (default) or portrait
outputsCount optional number Number of videos: 1-4 (default: 1)
startFrameBase64 optional string Base64 encoded image for image-to-video mode

Example: Text to Video

{
  "prompt": "A golden retriever running through a meadow at sunset",
  "aspectRatio": "landscape",
  "outputsCount": 1
}

Example: Image to Video 

{
  "prompt": "The scene comes alive with gentle wind movement",
  "mode": "frames-to-video",
  "aspectRatio": "landscape",
  "startFrameBase64": "data:image/jpeg;base64,/9j/4AAQ..."
}

Response 200 OK 

{
  "success": true,
  "videoUrls": [
    "https://storage.googleapis.com/ai-sandbox-videofx/video/..."
  ]
}

Error Response 4xx/5xx 

{
  "error": "Error message description"
}

üìã cURL Examples

Text to Video

curl -X POST http://localhost:3000/api/veo \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "prompt": "A cat playing with a ball",
    "aspectRatio": "landscape"
  }'

Image to Video (PowerShell) 

$base64 = [Convert]::ToBase64String([IO.File]::ReadAllBytes("image.jpg"))
$body = @{
  prompt = "The scene comes alive"
  mode = "frames-to-video"
  startFrameBase64 = "data:image/jpeg;base64,$base64"
} | ConvertTo-Json

Invoke-RestMethod -Uri "http://localhost:3000/api/veo" `
  -Method POST -Body $body `
  -ContentType "application/json" `
  -Headers @{"x-api-key"="YOUR_API_KEY"}
